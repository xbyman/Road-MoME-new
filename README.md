# Road-MoME: 基于多模态混合专家的路面异常检测系统

Road-MoME 是一个专门针对大规模路面场景设计的协同异常检测框架。它通过整合 3D 激光雷达（LiDAR）的几何信息与 2D 工业相机的纹理特征，利用**混合多模态专家 (MoME)** 架构，实现了对路面坑洼、裂缝及修补痕迹的高精度识别。

## 🌟 核心亮点

- **大规模处理能力**：支持 10,000+ 帧量级的多模态数据全自动对齐与特征提取。
- **混合专家系统 (MoME)**：
   - **物理专家 (Phys)**：利用 8 维硬性统计指标，确保对深度异常的绝对敏感。
   - **几何专家 (3D)**：基于 **Point-MAE** 提取拓扑特征，捕捉坑洼的破碎感。
   - **纹理专家 (2D)**：利用 **DINOv2** 提取高语义视觉特征，补位检测裂缝、油污等非结构性病害。

- **弱监督容错**：创新性地引入“质量感知门控 (Quality-aware Gating)”，能够有效吸收和纠正几何伪标签中约 40% 的噪声干扰。
- **轻量化部署**：针对民用级显卡（如 RTX 4060 8GB）进行了显存深度优化，支持断点续传。

## 🏗️ 系统架构

```
graph TD
    A[原始 PCD + 图像] --> B[数据对齐与索引]
    B --> C1[3D 预处理: RANSAC + ROI Patch]
    B --> C2[2D 预处理: 清晰度评估]
    C1 --> D1[物理特征: 8-dim Stats]
    C1 --> D2[几何特征: Point-MAE 384d]
    C2 --> D3[纹理特征: DINOv2 768d]
    D1 & D2 & D3 --> E[MoME 门控路由]
    E --> F[专家加权融合决策]
    F --> G[异常热力图输出]

```

## 🚀 快速开始

### 1. 环境准备

建议使用 Python 3.10，并安装必要的依赖：

```
conda create -n road_mome python=3.10
conda activate road_mome
pip install -r requirements.txt

```

### 2. 路径配置

在 `config/config.yaml` 中修改你的本地数据路径：

```
paths:
  raw_pcd_dir: "你的点云路径"
  raw_img_dir: "你的图片路径"
  output_dir: "./data/frame_packages"

```

### 3. 一键运行全流程

系统采用模块化设计，通过 `main.py` 调度执行：

```
python main.py

```

你可以在 `main.py` 中通过 `STEPS` 开关控制跳过已完成的部分（如特征提取）。

## 📂 脚本说明

| 脚本 | 功能描述 |
| --- | --- |
| `dataset_indexer.py` | 递归扫描各时间戳文件夹，建立 3D/2D 1:1 对齐索引。 |
| `0_master_preprocess.py` | 执行 RANSAC 扶平、排水坡补偿及 Patch 级物理特征计算。 |
| `1_extract_2d_dinov2.py` | 批量提取 DINOv2 视觉特征并评估图像置信度。 |
| `2_extract_3d_deep.py` | 利用 GPU 加速提取 Point-MAE 几何深度语义。 |
| `3_train_mome.py` | 启动弱监督训练，集成 TensorBoard 实时监控。 |
| `4_inference_vis.py` | 生成三栏对比诊断报告（预测图、标签图、专家权重分布）。 |

## 📊 实验观察

### 训练监控 (TensorBoard)

模型在万级数据量下展现了极佳的收敛性：

- **Train Loss**: 稳定下降至 0.004。
- **Val Loss**: 维持在 0.007 附近，泛化性能优越。

### 结果可视化

生成的“块状网格热力图”可直观展示：

1. **边缘保底**：路边缘自动切换至物理统计判定。
2. **中心信任**：行驶区域由 3D 几何特征主导。
3. **视觉唤醒**：在检测到裂缝或修补带时，2D 纹理专家自动获得更高权重。
